# AssetFlow Frontend

React + Vite + TailwindCSS frontend for AssetFlow ERP.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at http://localhost:3000

## Build

```bash
npm run build
```

## Features

- **Login/Signup Screen** with dark mode design
- Role-based authentication (Employee signup, Admin assigns roles)
- JWT token management
- Smooth animations and transitions
